== Description ==

Extend your website with interactive info blocks, call to action buttons or banners. Endless variation possibilities with more than 1o different layout support. Please note that this plugin should work out of the box though depending on the theme used you may need to adjust layout width and paddings. These settings are available under Container section of Inspector Control. 

== Installation ==

This plugin can be installed directly from your site.

1. Log in and navigate to _Plugins &rarr; Add New_.
2. Type "Kenzap CTAâ€ into the Search and hit Enter.
3. Locate the Kenzap CTA plugin in the list of search results and click **Install Now**.
4. Once installed, click the Activate link.
5. Go to Pages > Add New > Find CTA block.
6. Adjust Container > Max width setting if elements are not displayed properly. 


It can also be installed manually.

1. Download the Kenzap CTA plugin from WordPress.org.
2. Unzip the package and move to your plugins directory.
3. Log into WordPress and navigate to the Plugins screen.
4. Once installed, click the Activate link.
5. Go to Pages > Add New > Find Kenzap CTA blocks.
6. Adjust Container > Max width setting if elements are not displayed properly. 


== Frequently Asked Questions ==

1. What is this plugin for?
To extend your current theme with such elements as counter up circles, numbers, countdown elements. Refer to screenshots for more details

2. Why columns look narrow?
Depending on your front page container layout type you may need to reduce Max width setting in Inspector Control
